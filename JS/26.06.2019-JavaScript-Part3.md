## JavaScript for loop what will iterate from -1 to 10 and check the odd or even
```
var i;
for (i = -1; i < 11; i = i + 1 )
{
if (i % 2 !== 0 ) 
{
console.log (i + " => odd");
} else 
{
console.log (i + " => even")
}
}
```
